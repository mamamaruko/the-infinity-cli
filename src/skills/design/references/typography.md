# Typography Reference

## Purpose

Typography defines rhythm, hierarchy, and tone before color or effects.

## Hierarchy Baseline

- Display: 40-56px, tight tracking, used sparingly
- H1/H2: 28-40px, clear semantic hierarchy
- Body: 16-18px for primary reading
- Secondary text: 14-15px with strong contrast
- Labels/captions: 12-13px, avoid long paragraphs

## Spacing Rules

- Use a consistent vertical rhythm (4px or 8px grid)
- Larger jumps between sections than within sections
- Keep line length near 45-80 characters for paragraphs

## Readability Checks

- Paragraph line height: 1.4-1.7
- Heading line height: 1.1-1.3
- Avoid low-contrast thin weights for body copy

## Tone Mapping

- Premium: high contrast, generous whitespace, restrained weight changes
- Technical: clear monospace accents, dense but ordered sections
- Friendly: softer contrast, warm headings, simple sentence structure

## Anti-Patterns

- Too many font families in one surface
- Large headline with weak supporting hierarchy
- Tiny action text on touch targets
- All-caps without letter spacing
